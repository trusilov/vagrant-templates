# vagrant-templates

![](https://grantnorwood.com/app/uploads/2016/12/vagrant-logo.png)
*Start or create a VM*
```sh
 vagrant up
 ```
#####  *Restarting the VM*
```sh
vagrant reload
```
#####  *Stops the VM*
```sh
vagrant halt
```
#####  *Removes the VM*
```sh
vagrant destroy
```  
#####  *"freezes" the VM*
```sh
vagrant suspend
```  
#####  *Lists all previously created VMs in the host system*
```sh
vagrant global-status
``` 
#####  *View VM status*
```sh
vagrant status
```
